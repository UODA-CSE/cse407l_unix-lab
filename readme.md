## Objective

- To learn the basic Unix commands and shell scripting.

## How to create or edit a file

```bash
$ nano <filename>.sh
```

### Bash file structure

```bash
#!/bin/bash
echo "Hello World"
```

## How to run a file

```bash
$ nano <filename>.sh
$ bash <filename>.sh
```
